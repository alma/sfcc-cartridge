# sfcc-plugin

Alma plugin for SFCC

The full documentation is available [here](https://docs.getalma.eu/docs/cartridge-installation-guide).

## setup

```
npm i npm@6.14.11
npm i
```

create and fill your `.env` and `dw.json` files.

## create metadata
```
npm run build
```

## upload code
```
npm run uploadCartridge
```
